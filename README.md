## 📌 Overview

A simple, lightweight and modern Todo List application built using **HTML, CSS, and JavaScript**. 
This app allows users to add, delete, and manage daily tasks efficiently with a clean and responsive UI.

---

## ✨ Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 💾 Data persistence using browser storage (LocalStorage)
* 📱 Responsive design (works on mobile & desktop)
* ⚡ Fast and lightweight (no frameworks used)
---

## 🛠️ Technologies Used/Tech Stack

* HTML5 ((Semantic Markup) -> Structure of the app)
* CSS3 ((Flexbox, Grid, Responsive Design) -> Styling & Layout)
* JavaScript ((ES6+) -> logic and interactivity)
* LocalStorage API –> (Saving tasks in browser)

## 🧠 How It Works

* Uses **event delegation** for efficient button handling
* Updates display dynamically via DOM manipulation
* Validates input before calculation
* Evaluates expressions using a safer alternative to `eval()`

---

## ⚠️ Limitations

* 💾 Limited Data Storage (LocalStorage)
* 🌐 No Backend / Server
* 🔒 Basic Security
* ⚡ Scalability Issues
* 🌍 No Offline-to-Online Sync Logic
* 🧪 No Testing or Error Handling
